Project Initiation & Project Planning — Path'Ora
PHASE 1: PROJECT INITIATION
A. Hasil dan Ekspektasi Proyek
Proyek Path'Ora bertujuan membangun platform berbasis web yang membantu pencari kerja memahami profil kompetensinya secara otomatis dan mendapatkan rekomendasi jalur karir yang sesuai. Latar belakang proyek ini berasal dari permasalahan nyata yang masih banyak terjadi, yaitu kesenjangan antara kompetensi pencari kerja dengan kebutuhan industri, proses persiapan kerja yang masih manual dan subjektif, serta sulitnya perusahaan menyaring kandidat secara efisien.
Hasil yang diharapkan tercapai di akhir proyek ini adalah:
1. Platform Web Online yang Fungsional Platform Path'Ora dapat diakses secara publik dengan fitur utama upload CV, analisis otomatis, dan tampilan rekomendasi karir. Deployment dilakukan menggunakan Vercel, Netlify, atau Google Cloud Platform.
2. Model Machine Learning dengan Akurasi ? 85% Model klasifikasi kesiapan kerja dibangun menggunakan TensorFlow dan dievaluasi menggunakan metrik accuracy, precision, recall, dan F1-score. Model disimpan dalam format produksi (.keras / SavedModel) dan siap diintegrasikan ke backend.
3. Sistem NLP untuk Analisis CV Model NLP mampu melakukan parsing CV secara otomatis, mengekstraksi skill dan pengalaman, serta menghasilkan rekomendasi jalur karir yang personal menggunakan pendekatan LLM.
4. Dashboard Strategis dan Interaktif Dashboard menampilkan visualisasi hasil analisis CV, skill gap, dan rekomendasi karir secara informatif, dibangun menggunakan React.js dan Streamlit.
5. RESTful API yang Stabil Backend menyediakan API terstruktur yang menghubungkan frontend, model AI/ML, dan database secara terintegrasi.
